프로젝트 Our Travel 실행방법

DemoApplication를 실행합니다.
그리고 터미널로 들어가서 현재 위치에서 
cd my-app을 입력합니다.

제 로컬에서 프로젝트가 작동되는 위치는 PS C:\Users\윤찬익\Desktop\SW설계 Our Travel프로젝트 컴공_22212125_윤찬익\demo\my-app> 
이렇게 되어 있습니다.
그리고 
npm run dev
을 입력해주면 됩니다.
(ex 
PS C:\Users\윤찬익\Desktop\SW설계 Our Travel프로젝트 컴공_22212125_윤찬익\demo\my-app> npm run dev
)

웹 브라우저에 
http://localhost:3000
을 입력해주면 Our Travel 메인 페이지로 이동합니다.

로그인 할 수 있는 아이디는 
아이디: qwer
비밀번호: 1234
라고 입력하면 둘리2 이름으로 로그인 됩니다.

다른 방법으로는 회원가입을 직접 하셔도 됩니다.
데이터베이스는 aws에서 배포하여 작동하고 있기에 따로 하실 설정을 하실 필요는 없습니다.

행사 추천 카드에서 행사 둘러보기 버튼(분홍색)을 클릭하시면 오픈api에서 데이터를 받아와 현재 날짜를 기준으로 행사들이 나타납니다.
행사 둘러보기 페이지에서 자유롭게 검색 기능이나 날짜를 직접 선택하셔도 됩니다

여행지 추천 카드에서 여행지 탐색(초록색)을 클릭하시면 오픈 api에서 데이터를 받아와 이름 순으로 관광지들이 나열됩니다.
여행지 탐색 페이지에서 자유롭게 검색 기능이나 필터링 기능을 이용하시면 됩니다.

메인 페이지에서 위의 검색 창에 자유롭게 키워드를 입력하시고 검색(엔터)하시면 행사나 지역에 관한, 입력한 키워드에 관련된 데이터들이 나열됩니다.
로고 버튼을 누르시면 메인 페이지로 돌아옵니다.

팀원 찾기와 게시판 기능은 시간이 부족하여 아직 구현을 덜했습니다.

백엔드의 기능만을 보고 싶으시면
http://localhost:8080/swagger-ui/index.html
로 들어가시면 스웨거 페이지가 나와 백엔드 기능을 자세히 확인하실 수 있습니다.
